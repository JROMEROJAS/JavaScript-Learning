# ¿Que es JavaScript?
Es un lenguaje de programacion bastante utilizado en todo el ambito de la programacion. Con este lenguaje se puede trabajar como desarrollador web (frontend-backend o incluso desarrollador mobil) ademas de esto, tambien se puede trabajar con microcontroladores e incluso apps de escritorio.

# Framework vs Libreria
Un framework te entrega herramientas con el fin de poder desarrollar aplicaciones complejas. Por otra parte, una libreria se encarga de solucionar un problema en especifico.

# ¿Que es ECMAScript?
Cuando se creo JavaScript se hizo principalmente para que se ejecutara dentro de los exploradores web, esto se hizo porque para aquel entonces los sitios web eran estaticos, es decir, si tu querias ver contenido nuevo dentro de una pagina web tu tenias que navegar a otra pagina web. Luego de esto los creadores de JS decidieron entregarle este proyecto a la organizacion de ECMA, esta empresa ha sido bastante popular por el apoyo que ha tenido en la realizacion de diferentes proyectos, tales como:

- **ECMA 334 (En este proyecto contribuyeron al desarrollo de C#).**
- **ECMA 404 (En este proyecto contribuyeron al desarrollo de JSON).**
- **ECMA 408 (En este proyecto contribuyeron al desarrollo de Dart).**
- **ECMA 262 o tambien conocido como ECMAScrip esto indica los estandares para los lenguajes de programacion JS.**

En la actualidad la mayoria de navegadores por no decir todos, tienen un motor que se encarga de ejecutar JS. Aqui entrataria un dato particular que es el motor V8 el cual posee Google Chrome y es codigo abierto, ahora bien, gracias a esto un desarrollador llamado **Ryan Dahl** pudo embeber el motor V8 dentro de codigo hecho por C++, al hacer esto es por eso que JS puede ser ejecutado en muchos ambitos. Este codigo escrito en C++ que permitio embeber el motor V8, es el conocidicimo **Node.js**.
