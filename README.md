# JavaScript Learning - 5to Semestre

Repositorio dedicado a documentar mi proceso de aprendizaje en JavaScript, reforzando las bases de lógica de programación y preparándome para la integración con frameworks modernos y entornos Backend.

##  Objetivos
- Dominar la sintaxis moderna (ES6+).
- Entender el asincronismo para conexiones con APIs (Spring Boot).
- Aplicar lógica de resolución de problemas mediante ejercicios prácticos.

## Estructura del Repositorio

| Módulo | Contenido | Estado |
| :--- | :--- | :--- |
| **01-Fundamentos** | Tipos de datos, variables y operadores. | ⏳ En proceso |
| **02-Estructuras** | Condicionales, bucles, arreglos y objetos. | ⏳ Pendiente |
| **03-Funciones** | Arrow functions, scopes y callbacks. | ⏳ Pendiente |
| **04-Asincronismo** | Promesas, Async/Await y Fetch API. | ⏳ Pendiente |
| **05-DOM** | Manipulación del navegador con JS. | ⏳ Pendiente |
