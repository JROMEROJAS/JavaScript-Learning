#  JavaScript Learning

Este repositorio documenta mi proceso de aprendizaje en JavaScript.

El objetivo principal es reforzar las bases de la programación, mejorar mi lógica y prepararme para el uso de frameworks modernos y desarrollo backend.

---

##  Objetivos

- Comprender la sintaxis moderna de JavaScript (ES6+).
- Aplicar lógica de programación mediante ejercicios prácticos.
- Entender el manejo del asincronismo (promesas, async/await).
- Prepararme para la integración con tecnologías como Spring Boot (APIs REST).
- Desarrollar buenas prácticas en código limpio y organizado.

---

##  Contenidos de Aprendizaje

A lo largo del repositorio estaré trabajando los siguientes temas:

- Tipos de datos y variables
- Operadores
- Condicionales
- Bucles (for, while)
- Funciones (tradicionales y arrow functions)
- Arrays y objetos
- Manipulación del DOM
- Eventos
- Asincronismo (callbacks, promesas, async/await)
